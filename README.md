# Custom Animated Calculator

A stylish and interactive calculator featuring social media integration and smooth animations. This project was developed to demonstrate proficiency in DOM manipulation, GSAP library usage, and modern layout techniques.

## Features

- <b>Interactivity</b>: Every button provides visual feedback upon clicking.

- <b>GSAP Animations</b>: Smooth rotation and scaling effects for the avatar, along with entrance animations for the buttons.

- <b>Custom Design:</b> GitHub and LinkedIn icons integrated directly into the calculator interface in place of standard buttons.

- <b>Responsiveness:</b>Fluid display across various screen sizes achieved through relative units (REM/EM/VH/VW).

## Tech Stack

<b>HTML5</b> – Semantic structure of the application.

<b>CSS3</b> – Advanced styling using Flexbox/Grid and absolute positioning for precise image placement.

<b>JavaScript</b> – Core logic for mathematical operations and calculations.

<b>GSAP (GreenSock Animation Platform)</b> – Powering complex, high-performance UI animations.

<b>Remix Icon</b> – A comprehensive library of modern system icons.
